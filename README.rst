airflow-provider-xlsx
=====================

Airflow operators for converting XLSX files from/to Parquet/CSV


System Requirements
-------------------

* Airflow Versions
    * 2.0.0 or newer

Installation
------------
::

    pip install airflow-provider-xlsx
